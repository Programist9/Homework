Какие команды использовали в терминалена на уроке

  • "git --version" проверка версии гита
  • "cd .." выйти на уровень верх
  • "ls" посмотреть какие есть папки
  • "git clone 'ссылка'" клонирование папки с Github на пк
  • "git log" проверка изменений
  • "git status" почти как прошлое
  • "git push origin main"
  • "git commit -m "название""
  • "git add ."
  • "git checkout -b first-homework"
  • "git pull"
